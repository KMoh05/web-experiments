# GEE Building Analysis

This project contains JavaScript code for Google Earth Engine (GEE) to analyze building data within a specified area.

## Features
- Filter building polygons by confidence levels
- Visualize buildings with color coding
- Calculate total number of buildings within a drawn area

## How to use
1. Copy the JavaScript code from `main.js`.
2. Visit [Google Earth Engine Code Editor](https://code.earthengine.google.com/) and sign in.
3. Paste the code into the script editor.
4. Use the "Draw a shape" tool to select your area.
5. Run the script to see the total number of buildings printed in the console.
